# ntistools

R package to wrangle Nonprofit Trends and Impacts Survey data.

## Installation

Install from GitHub with:

```r
# install.packages("remotes")
remotes::install_github("UrbanInstitute/ntistools")
```

To include the vignette:

```r
remotes::install_github("UrbanInstitute/ntistools", build_vignettes = TRUE)
```
