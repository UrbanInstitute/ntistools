# ntistools: Tools for Analyzing Nonprofit Trends and Impacts Survey Data

Reusable functions for wrangling, analyzing, and visualizing Nonprofit
Trends and Impacts Survey (NTIS) data. This package is for internal use
only and does not contain any raw survey data or PII.
